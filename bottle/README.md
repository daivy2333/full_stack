# 漂流瓶项目

这是一个使用现代JavaScript技术重构的漂流瓶应用，采用模块化架构，代码结构清晰，易于维护和扩展。

## 项目结构

```
bottle/
├── index.html          # 主HTML文件
├── styles.css          # 样式文件
├── bottles.json        # 漂流瓶数据
├── js/                 # JavaScript目录
│   ├── main.js         # 主入口文件
│   └── modules/        # 模块目录
│       ├── appController.js    # 应用控制器
│       ├── bottleService.js     # 漂流瓶服务
│       ├── storageService.js    # 存储服务
│       ├── uiController.js      # UI控制器
│       └── utils.js             # 工具函数
└── README.md           # 项目说明文档
```

## 模块说明

### 1. appController.js
应用的核心控制器，负责协调各个模块，处理应用的核心业务逻辑。

### 2. bottleService.js
漂流瓶服务模块，负责处理漂流瓶的数据操作，包括加载、保存、获取等。

### 3. storageService.js
存储服务模块，负责处理所有与本地存储相关的操作。

### 4. uiController.js
UI控制器模块，负责处理所有UI相关的操作，包括DOM操作、事件监听等。

### 5. utils.js
工具模块，包含通用的工具函数。

### 6. main.js
主入口文件，负责初始化应用。

## 技术特点

1. **模块化架构**：使用ES6模块系统，将代码划分为多个功能模块，提高代码的可维护性和可复用性。

2. **面向对象设计**：使用类和对象来组织代码，使代码结构更加清晰。

3. **关注点分离**：将数据操作、UI操作和业务逻辑分离，降低代码耦合度。

4. **现代JavaScript特性**：使用ES6+特性，如箭头函数、解构赋值、模板字符串等。

5. **错误处理**：添加了适当的错误处理机制，提高应用的健壮性。

## 如何使用

1. 直接在浏览器中打开`index.html`文件即可使用应用。

2. 如需修改或扩展功能，可以编辑相应的模块文件。

3. 如需添加新的工具函数，可以在`utils.js`中添加。

点击标题5次可以解锁开发者模式，开发者模式下可以绕过每日限制，方便测试。

## 注意事项

1. 本项目使用了ES6模块系统，需要在支持ES6模块的现代浏览器中运行。

2. 数据存储在浏览器的localStorage中，清除浏览器数据会导致数据丢失。

3. 本项目仅用于学习和演示目的，不包含后端服务。
